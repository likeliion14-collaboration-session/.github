# 🦁 서경대학교 멋쟁이 사자처럼 14기 PO-PE_BE 연합 세션 🦁
### 이곳은 서경대학교 멋쟁이 사자처럼 전체 파트 연합세션을 위한 Organization입니다.

---

## 📍 프로젝트 소개

**연동된 사람과 실시간으로 위치·사진을 공유하며 하루의 동선을 기록하고, 원하는 순간 "오늘의 동선 카드"로 뽑아 SNS에 공유하는 웹앱**입니다.

사용자는 "기록 시작" 버튼을 누른 순간부터의 이동 경로와 촬영한 사진을 지도 위에 자동으로 기록하고, 그 기록을 카드 형태로 만들어 다운로드하거나 공유할 수 있습니다. 초대 코드를 통해 다른 사용자와 연동하면, 서로의 동선과 사진을 실시간으로 확인할 수도 있습니다.

세미 해커톤 주제 중 **"③ 자발적인 공유와 확산"**을 기반으로 기획되었습니다.

## 🤔 기획 배경 / 문제 상황

- SNS에 무언가를 올릴 때마다 사진을 고르고 편집하는 과정에 부담을 느끼는 사람이 많습니다. 반면 자동으로 만들어진 결과물(예: Spotify Wrapped)은 부담 없이 공유하고 싶어지는 경향이 있습니다.
- 하루 동안 어디를 다녔고 어떤 사진을 찍었는지는 시간이 지나면 기억에서 흐려지고, 여러 앱(사진첩, 지도 앱 등)에 흩어져 있어 한눈에 되돌아보기 어렵습니다.
- 기존의 위치 기반 서비스들은 "지역을 수집하고 채워나가는" 컬렉션형 게임 요소에 집중된 경우가 많아, "오늘 하루를 기록하고 공유한다"는 목적에 맞는 서비스는 상대적으로 드뭅니다.
- 연인이나 친구와 떨어져 있어도 서로의 하루를 자연스럽게 느끼고 싶어하는 니즈가 있지만, 이를 위한 가벼운 실시간 공유 수단은 마땅치 않습니다.

이러한 문제를 해결하기 위해, **이동하고 사진을 찍는 일상적인 행동만으로 자동으로 공유 콘텐츠가 만들어지는 서비스**를 기획했습니다.

## ✨ 주요 기능

- **닉네임 기반 로그인**: 별도 비밀번호 없이 닉네임(+선택적으로 프로필 이미지)으로 계정 생성 및 로그인
- **실시간 동선 기록**: "기록 시작" 이후 일정 주기로 위치를 자동 기록, 지도 위에 현재 위치와 이동 경로 표시
- **사진 업로드 및 위치 핀**: 사진 업로드 시 EXIF에서 위치·촬영시각을 자동 추출, 지도 위 위치에 핀으로 표시 (코멘트·아이콘 첨부 가능)
- **오늘의 동선 카드 발급**: 지금까지의 동선과 대표 사진(최대 3장, 선택 안 해도 됨)을 조합해 카드 생성, 이미지로 다운로드/공유
- **초대 코드 기반 연동**: 6자리 초대 코드를 통해 다른 사용자와 연동, 인원 제한 없음
- **공유 범위 제어**: 연동된 상대방마다 개별적으로 내 기록 공유 여부를 켜고 끌 수 있으며, 끈 시점 이전 기록까지는 계속 공유됨

## 🔄 서비스 흐름

```
닉네임 입력 (로그인 대체)
   │
   ▼
"기록 시작" 클릭 (하루 1회)
   │
   ├─▶ 일정 주기로 위치 자동 기록 → 지도에 현재 위치·이동 경로 표시
   │
   └─▶ 사진 업로드 (코멘트·아이콘 첨부) → 지도에 위치 핀 표시
   │
   ▼
"카드 만들기" 클릭
   │
   ├─▶ 지금까지의 동선 + 사진 후보 조회
   ├─▶ 대표 사진 최대 3장 선택 (선택 안 해도 됨)
   └─▶ 카드 확정 발급 → 이미지로 다운로드 / SNS 공유
   │
   ▼
(선택) 초대 코드 생성 → 상대방이 코드 입력 → 연동 성사
   │
   ├─▶ 연동 목록에서 상대방 동선·사진 확인
   └─▶ 상대방별로 내 기록 공유 여부 개별 설정 (켜기/끄기)
```

## 📑 API 명세서

### 사용자 (User)
| Method | 경로 | 기능 |
|---|---|---|
| POST | `/users/login` | 닉네임 입력 (로그인 대체, 없으면 신규 생성) |
| POST | `/records/start` | 오늘의 동선 기록 시작 |

### 동선 (Track)
| Method | 경로 | 기능 |
|---|---|---|
| POST | `/locations` | 현재 위치 저장 (실시간 위치 + 동선 이력 동시 갱신) |
| GET | `/track-points` | 특정 날짜의 동선 좌표 목록 조회 |
| GET | `/users/live-location` | 현재 위치 조회 |

### 사진 (Photo)
| Method | 경로 | 기능 |
|---|---|---|
| POST | `/photos` | 사진 업로드 (EXIF 위치·시각 자동 추출) |
| GET | `/photos/pins` | 지도 표시용 사진 핀 목록 조회 |
| GET | `/photos/{photoId}` | 사진 상세 조회 |
| PATCH | `/photos/{photoId}` | 사진 코멘트·아이콘 수정 |
| DELETE | `/photos/{photoId}` | 사진 삭제 |

### 카드 (Card)
| Method | 경로 | 기능 |
|---|---|---|
| GET | `/card/candidates` | 카드 대표 사진 후보 및 동선 조회 |
| POST | `/card` | 오늘의 동선 카드 확정 발급 |

### 초대 (Invite)
| Method | 경로 | 기능 |
|---|---|---|
| POST | `/invites` | 초대 코드 생성 |
| POST | `/invites/accept` | 초대 코드 입력하여 연동 |

### 연동 및 공유 (Connection / Share)
| Method | 경로 | 기능 |
|---|---|---|
| GET | `/users/{userId}/connections` | 연동된 사용자 목록 조회 |
| PATCH | `/connections/{connectionId}/sharing` | 특정 상대방에 대한 공유 켜기/끄기 |
